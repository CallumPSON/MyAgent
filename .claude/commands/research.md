Research a topic using web search and return a concise, cited summary.

Steps:
1. Use $ARGUMENTS as the search query. If no arguments, ask the user what to research.
2. Perform web searches to gather information from multiple sources.
3. Fetch and read the most relevant pages.
4. Synthesize findings into a structured response:
   - **Summary** — 2-3 sentence answer to the query
   - **Key findings** — bullet points of the most important facts
   - **Sources** — list the URLs consulted
5. If the research is for a coding task (e.g. "how to use library X"), include a minimal working code example.
6. Flag any conflicting information found across sources.

Keep the response focused and under 400 words unless the topic requires more depth.
