Set or display the current session goal.

If arguments are provided, treat them as the new goal statement and confirm it is set.
If no arguments are provided, display the current goal (stored in the conversation context) or say no goal is set.

Usage:
- `/goal <condition>` — set the goal, e.g. `/goal get all tests passing`
- `/goal` — show the current goal

When a goal is set, briefly confirm it and remind yourself to check progress against it before ending each turn.
When showing the current goal, also report whether it has been achieved based on the conversation so far.
